class decision_rules.serialization.utils.JSONClassSerializer

Bases: ABC

Abstract class for classes serializer. Each serializer should inherit this class and be registered for type by “register_serializer”. Each class serializer should have inner class “Model” which is pydantic model class for JSON representation.
